# APT - Advanced Packaging Tool

**Software para gerenciamento de pacotes no Debian e sua variantes.**

### O arquivo /etc/apt/sources.list

Lista as fontes de onde ele obterá os pacotes.

- Exemplos de entradas

    deb http://deb.debian.org/debian/ trixie main non-free-firmware  
    deb-src http://deb.debian.org/debian/ trixie main non-free-firmware

A primeira palavra das linhas indica indica se guarda pacotes binários (deb), ou seja, pré-compilados ou pacotes fonte (deb-src).  
Sempre que esse arquivo for modificado, deve se rodar o comando apt-get update, para que o APT obtenha as listas de pacotes das fontes especificadas.

- Abrir arquivo: `nano /etc/apt/sources.list `
- Atualizar listas de pacotes `apt update`
- Adicionar DVD no source.list `apt-cdrom add`
- Instalar pacote: `apt install [pacote]`
- Remover pacote: `apt remove [pacote]`

### APT - Opções

- `apt clean`: Apaga todos os pacotes baixados e já instalados.
- `apt autoclean`: Apaga todos os pacotes que não possam mais ser baixados.
- `apt autoremove`: Remove pacotes órfãos.
- `apt install -d [pacote]`: baixar arquivo sem instalar.
- `apt-cache dump`: Lista todos os pacotes instalados.
- `apt-cahche stats`: Mostra informações como o número de pacotes intslados e de dependências.
- `apt install -f`: Verifica lista de depedências quebradas e tenta corrigi-las instalando pacotes necessários.
- `apt purge [pacote]`: Remove pacotes incluindo seus arquivos de configuração.
- `apt upgrade`: Atualiza os pacotes do sistema.
- `apt full-upgrade`: Atualiza a distribuição para uma nova.
