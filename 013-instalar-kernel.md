# Instalar novo Kernel no sitema Linux Debian

***FAÇA BACKUP DOS SEUS ARQUIVOS ANTES***

### Comandos utilizados

 - `uname -r`: Mostra a versão atual do Kernel.
 - `apt-cache search linux-image`: Mostra as versões de kernel disponíveis para a distribuição utilizada.
 - `apt install [imagem_escolhida]`: Instala o novo kernel.
 - `shutdown -r now`: Reinicia o sistema.
