# RAID 5 no Linux
## Configuração e adição ao FSTAB

### RAID 5
Mínimo 3 discos
/dev/sdb
/dev/sdc
/dev/sdd

1. Criar partições específicas para RAID nos discos  
`fdisk -l` visualizar discos  
`fdisk /dev/sdb`  
`d` excluir partição  
`n` nova partoção  
`t` muda tipo da partição código `fd` para RAID.

2. Criar a matriz RAID  
`mdadm --create --force --assume-clean /dev/md0 --verbose --level=5 --raid-devices=3 /dev/sd[bcd]1`  

3. Adicionar o array ao arquivo de configuração    
`cat /etc/mdadm/mdadm.conf`  
`mdadm -Es | grep md >> /etc/mdadm/mdadm.conf`  
`cat /etc/mdadm/mdadm.conf`  

4. Formatar o array:  
`mkfs.ext4 /dev/md0`   

5. Montar o dispositivo:  
`mount /-t ext4 /dev/md0 /raid`  

6. Acrescentar o array ao fstab (p/ montar automaticamente na inicialização)  
`blkid` Descobrir o UUID do array  
`nano /etc/fstab` Adicionar ao final do arquivo:  
UUID=id_encontrado /raid ext4 defaults 0 0  (dump=0, pass=0)  
`df -h` mostra dispositivo montado
