# Alguns comando que não necessárimente foram mostrados no curso:

| Comando   | Função |
| --------- | ------ |
| ip addr   | Semelhante ao ifconfig, mas não precisa instalar pacote externo |
|hostname -I|Mostra IP|
|ssh user@ip|Acesso ssh via terminal|
|su         |Entra no usuário root, equivalente 'sudo su' no mint|
|exit       |Sair|
|umask      |Subtrai permissões de arquivos ou diretórios.|
