# RAID por software no Linux 02
## Simular falhas, reparar e destruir array

Verificar o status do array:  
`mdadm --detail /dev/md0`  

`cat /proc/mdstat`  

Simular falha no array (por software):  
`mdadm --manage --set-fault /dev/md0 /dev/sdc1` 

Verificar status:  
`mdadm --detail /dev/md0`  

Remover um disco com falha:  
`mdadm /dev/md0 -r /dev/sdc1`  

Adicionar disco substituto ao array:  
`mdadm /dev/md0 -a /dev/sdc1`  
`mdadm --detail /dev/md0`  

Excluir array RAID  
Parar o array:  
`mdadm --stop /dev/md0`  
Remover:  
`mdadm --remove /dev/md0`  
Excluir o superbloco em todos os drives do array:  
`mdadm --zero-superblock /dev/sd[bc]1`  
Editar o arquivo /etc/mdadm/mdadm.conf  
Excluir partição, criar nova ...   
`fdisk /dev/sdb`
