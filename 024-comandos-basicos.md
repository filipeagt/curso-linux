# Comandos básicos do Linux  07
## cp (copiar arquivos), rm (remover arquivos)

### Comando cp
Copia um arquivo ou diretório para outro local.  
`cp [opções] origem destino`  
Exemplos:  
`cp /home/filipe/arq1 /home/roberto`  
Opções:  
`-i` Pede confirmação interartivamente (s/n)  
`-r` Copia diretórios recursivamente  
`-p` Preserva todas as informações  

### Comando rm
Este comando pode ser utilizado para excluir tanto arquivos quanto diretórios.
`rm [opções] arquivo_diretório` Para excluir arquivos  
`rm -r nome_diretório` Para excluir um diretório ocupado.
